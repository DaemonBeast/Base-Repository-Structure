# Base Repository Structure

The base structure I use for my GitHub projects.

```
┣━ ACKNOWLEDGEMENTS.md              Copyright notices and licences of dependencies
┣━ CHANGELOG.md                     Software change log
┣━ CODE_OF_CONDUCT.md               Code of conduct
┣━ CONTRIBUTING.md                  Contributing guide
┣━ CONTRIBUTORS.md                  List of contributors
┣━ LICENSE                          License
┣━ README.md                        Project overview
┣━ SECURITY.md                      Vulnerability report guide
┣━ SUPPORT.md                       Support guide
┣━ .github                          GitHub files
┃  ┣━ CODEOWNERS                    Code owners
┃  ┣━ PULL_REQUEST_TEMPLATE.md      Pull Request template
┃  ┗━ ISSUE_TEMPLATE                Issue templates
┃     ┣━ ask-a-question.md          Question template
┃     ┣━ bug-report.md              Bug report template
┃     ┗━ feature-request.md         Feature request template
┣━ build                            Compiled files
┣━ doc                              Documentation
┣━ res                              Static files
┣━ scripts                          Build and test scripts
┣━ src                              Source files
┃  ┣━ index.js                      Entry file
┃  ┣━ package.json                  Package metadata
┃  ┗━ lib                           Main files
┃     ┗━ core.js                    Main file
┗━ test                             Test scripts
```
